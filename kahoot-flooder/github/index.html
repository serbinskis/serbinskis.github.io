About

Kahoot Flooder is a web-based tool to join automated bots into Kahoot quiz games using only a PIN, nickname prefix, and count. It operates via socket.io, runs on both HTTP and HTTPS, and uses the kahoot.js-latest library to mimic actual player behavior.
